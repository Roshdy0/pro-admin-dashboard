# 📊 Modern Admin Dashboard - Glassmorphism UI

A high-performance, professional admin dashboard built with a focus on modern UI/UX principles. This project features a sophisticated **Glassmorphism** design, interactive data visualization, and a fully responsive architecture optimized for all screen sizes.

## ✨ Features
*   **Modern UI:** Sleek "Dark & Light" gradient aesthetics with glassmorphism effects.
*   **Interactive Charts:** Dynamic data representation using **Chart.js**.
*   **Fully Responsive:** Seamless experience across Mobile, Tablet, and Desktop.
*   **SPA Navigation:** Smooth, fast page switching without reloading (Dynamic Sidebar).
*   **Clean Architecture:** Organized code structure, ready for **React.js** migration.

## 🛠️ Tech Stack
*   **Structure:** HTML5 (Semantic Tags)
*   **Styling:** CSS3 (Flexbox/Grid & Bootstrap)
*   **Logic:** Vanilla JavaScript & jQuery
*   **Visuals:** Font Awesome 7 & Chart.js

*Visual representation of the dashboard across multiple devices.*

## 📂 Project Structure
- `index_3.html`: Core structure of the dashboard.
- `style_2.css`: Professional styling and gradient configurations.
- `script_2.js`: Navigation logic and interactive elements.
- `js/char.js`: Configuration for charts and data visualization.
